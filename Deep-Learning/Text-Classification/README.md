# Projeto

## Classificação Binária (IMDB)
Modelo para análise de sentimentos em textos, explorando camadas densas e ativação sigmoid.

## Classificação Multiclasse (Reuters)
Categorização de notícias em 46 tópicos distintos, utilizando softmax e tratamento de dados categóricos.

## Fundamentos 
Técnicas de vetorização, regularização e monitoramento de métricas de perda (loss) e acurácia.

## Ferramentas 
- Python
- TensorFlow/Keras
- NumPy
